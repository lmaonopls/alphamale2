Linux - ი, როგორც ცნობილია, ერთ-ერთი ოპერაციული სისტემაა, რომელიც მიეკუთვნება ეგრედ წოდებულ open source ოპერაციულ სისტემათა ოჯახს. იგულისხმება, რომ მისი გამოყენება შეუძლია ნებისმიერ მომხმარებელს ნებისმიერი მიზნით, შეცვალოს ან გაუკეთოს მოდიფიკაცია სისტემას. Linux - ის პირველი ოპერაციული სისტემა ოფიციალურად გამოვიდა 1991 წლის 17 სექტემბერს ლინუს ტორვალდსის მიერ.

როგორც უკვე ავღნიშნეთ, Linux - ი open source სისტემაა და მისი გამოყენება ნებისმიერი მიზნით არის შესაძლებელი, სმარტფონები და სხვა სახის ტექნოლოგიები Linux - ის მოდიფიცირებული სისტემის საფუძველზე მუშაობს. დაცვის მხრივ Linux - ს უპირატესობა უჭირავს, რადგანაც ის ყველასთვისაა ხელმისაწვდომი, დეველოპერები მსოფლიოს სხვადასხვა კუთხიდან თვალს ადევნებენ და უმკლავდებიან სხვადასხვა ტიპის ვირუსსა თუ საფრთხეს.

ასევე ერთ-ერთი დადებითი თვისება, რომელიც ახასიათებს მოცემულ ოპერაციულ სისტემას, არის სისწრაფე, სიმსუბუქე და კომპაქტურობა. იგი მუშაობს თუნდაც ძალიან დაბალ მონაცემებზე, არ მოითხოვს დიდი რაოდენობით მეხსიერებას, რადგან არ საჭიროებს ანტივირუსის პროგრამის დაყენებას. Linux - ი ინტერფეისის მხრივაც საკმაოდ სიმპლისტურია, რაც საშუალებას აძლევს გაუმარტივოს მომხმარებელს საქმე.

მსგავსი ოპერაციული სისტემების, რომლებიც open source ოჯახს მიეკუთვნებიან, სწავლა და გამოყენება ბევრად ძნელია ვიდრე მაგალითად Windows -ისა ან Apple - ის. სისტემის აგებულება განსხვავებულია ამიტომაც მის სასწავლად დროა საჭირო. ზოგიერთი აპლიკაცია, მაგალითად ავიღოთ რაიმე ცნობილი ვიდეო თამაში, რომელიც მუშაობს ჩვენთვის ცნობილ Windows - ა თუ Apple - ის ოპერაციულ სისტემაში, არ მუშაობს Linux - ში, ის არ არის მხარდაჭერილი, რადგან Linux - ი ძირითადად სხვა მიზნებისთვისაა გათვლილი.

Linux - ი შეიძლება გამოვიყენოთ, როგორც სამუშაო სადგურად, ასევე სერვერის პლატფორმად. Linux-ის მომხმარებელთათვის არსებობს თანამედროვე გრაფიკული და საოფისე პროგრამები, ასევე მათთვის ინტერნეტში სრულყოფილად სამუშაოდ უამრავი პროგრამა, რომლითაც შესაძლებელია ვებ-გვერდების დათვალიერება, ელექტრონული ფოსტის მიღება და გაგზავნა.  არანაკლები რაოდენობით არსებობს პროგრამირებით დაინტერესებულ პირთათვის განკუთვნილი პროფესიონალური დონის გრაფიკული ინტერფეისიანი პროგრამული პროდუქტები.  Linux-ის მიზანია მომხმარებელს უფასოდ მიეწოდოს რაც შეიძლება მეტი ინფორმაცია და პროდუქტი (პროგრამა), რაც ხელსაყრელი ალტერნატივაა.
